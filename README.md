## Descripción

Este repositorio contiene mi portfolio personal desarrollado con **Astro**.  
Lo uso para practicar con este framework, organizar mis proyectos y mostrar mi trabajo como desarrollador web.

La idea principal es tener una página rápida, ligera y centrada en contenido real, no solo en una plantilla vacía.

---

## Características

- Sitio estático construido con **Astro**.
- Secciones para presentarme y contar quién soy.
- Listado de proyectos con enlaces a demos y repositorios.
- Diseño pensado para verse bien tanto en móvil como en escritorio.
- Código organizado en componentes dentro de `src/`.

---

## Tecnologías utilizadas

- **Astro**
- **JavaScript**
- **TailwindCSS**

## Scripts disponibles

Desde la carpeta del proyecto:

```bash
pnpm install
pnpm dev
pnpm dev → Levanta el servidor de desarrollo en http://localhost:4321.

pnpm build → Genera la versión de producción dentro de la carpeta dist.

pnpm preview → Sirve el build para comprobar cómo quedará antes de desplegar.

```

## Cómo ejecutarlo en local

Clona este repositorio:

```bash
git clone https://github.com/Matusbh/Portfolio-astro.git
cd Portfolio-astro
```

Instala las dependencias:

```bash
pnpm install
```

Inicia el entorno de desarrollo:

```bash
pnpm dev
```
